Sobre Mí

Hola, soy desarrollador .NET de El Salvador, C. A. trabajando en desarrollo web y otras tecnologías. Disfruto convirtiendo problemas complejos en soluciones simples. Mi trabajo es construir sus próximos sitios web, aplicaciones web y páginas web o trabajar con un proyecto existente que ya tenga. Agrego un toque personal a su producto y me aseguro de que sea llamativo y fácil de usar. >Tengo experiencia en tecnologías web de Microsoft como: ASP.NET Core, desarrollando API's RESTful, proyectos MVC y aplicaciones SPA (Blazor WebAssembly y Angular 12). Además, tengo experiencia en el uso del lenguaje de programación C# en aplicaciones de consola y Windows Forms. ¿Qué pasa con las bases de datos? bueno, he trabajado con motores SQL Server, PostgreSQL y MongoDB para crear esquemas de bases de datos complejos.
